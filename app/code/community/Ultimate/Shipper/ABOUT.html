Ultimate Multi Flat Shipping

Ultimate Multi Flat Shipping is a solution that allows to create unlimited flat rate methods for shipping.

Ultimate Multi Flat Shipping is a extension that overcomes the single flat rate option in magento by serving mutliple flate rate methods based on different prices and conditions without disturbing the system files and also with all new cool features. You can mention your own shipping carrier and unlimited methods under it. It also enables you to provide custom free shipping option. For more explained features see below.

Installation Methods

Direct/Manual Installation:

Step 1: Extract the zip file to a flexible location.

Step 2: Copy the folders app and media alone to your Magento project root directory, It will ask you for merging folders and files. Proceed with that.

Step 3: Login to your admin panel of your Magento project.

Step 4: Go to System->Cache Management.

Step 5: Select all options and in the actions with refresh selected, click submit.

Step 6: Flush Magento Cache, You will be now seeing the installed extension.

For Other Types of Installation

Proceed with magento connect.

IMPORTANT: Disable your cache before installing this extension!

Main Features

  • * Complete Backend Configuration: No need of any developers to code in.
  • * Unlimited allowed methods with different prices and conditions.
  • * Select allowed methods at ease in the admin section of shipping methods.
  • * Allowed methods with calculate handling fee, type and applied option.
  • * When allowed method is disabled and show message feature to all individual methods.
  • * Custom Free Shipping method with enable/disable and minimum order amount feature.
  • * Restrict other allowed methods if free shipping criteria is met feature.
  • * And all system flate rate features.
